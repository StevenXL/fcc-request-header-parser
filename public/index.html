API Basejump: WhoAmI Microservice

This is a whoami microservice. The purpose of this microservice is to return a JSON object with the ip, language, and software properties. These properties will reflect the relevant information from the client.

Example Usage

https://rocky-cliffs-9728.herokuapp.com/api/whoami

Example Output

{"ipaddress":"127.0.0.1","language":"en-US","software":"X11; Ubuntu; Linux x86_64; rv:43.0"}
